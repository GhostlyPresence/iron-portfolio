Forest Paths

A collection of small Rust exercises, each written like a walk along a path through a forest of data structures.

Forest Paths is my learning ground for Rust. Each module focuses on a single idea — ownership, iterators, trees, graphs — and frames it as a small journey with a beginning and an end.

The project includes both command‑line examples and unit tests, so I can quickly revisit concepts later. Over time I’d like this to become a reference trail I can hand to other learners who enjoy narrative explanations.
