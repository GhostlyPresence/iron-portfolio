macOS Three Finger Window Switch

Bringing macOS gestures to GNOME Shell — seamless window management for Linux.

The Challenge

Coming from macOS, one of the most intuitive features is the three-finger swipe gesture for switching between open windows. On Linux with GNOME Shell, this familiar workflow wasn't available. The goal was to recreate that seamless experience and bring the intuition of macOS window management to the Linux desktop.

The Solution

This GNOME Shell extension intelligently tracks all open windows and automatically maximizes them while moving them between workspaces. It works in harmony with GNOME's native window management system, providing a smooth and predictable experience that feels natural to anyone who's used macOS.

The extension is built with JavaScript and integrates directly with GNOME Shell's windowing system. It supports GNOME Shell 46+, ensuring compatibility with modern Linux desktops while maintaining minimal overhead.

Key Features

  • Maximize to Workspace: Windows automatically maximize when moving between workspaces for optimal usage of screen real estate
  • Intelligent Window Tracking: Monitors all open windows and their states in real-time
  • Seamless Integration: Works smoothly with GNOME Shell's native window management without conflicts
  • Zero Configuration: Install and it just works — no complex settings needed

Technology Stack

  • JavaScript (100% of the codebase)
  • GNOME Shell Extension API
  • Modern Linux desktop integration

Open Source Philosophy

This project is freely available on GitHub and welcomes contributions from the community. It represents a commitment to making the Linux desktop experience smoother and more intuitive for users transitioning from other operating systems.
